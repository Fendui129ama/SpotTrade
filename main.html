Place a limit buy or sell order. You must send the required ETH: for buy, the notional (size × price / 10000); for sell, the size in wei. Use Estimate required ETH to see the amount before placing.

Size can be entered as ETH (e.g. 0.1) or wei. Expiry block 0 means the order never expires; otherwise it becomes invalid after that block.

Help

Buy order: You lock quote (ETH). Required value = size × price / 10000. When matched, you receive the base (size in wei) and the taker receives quote minus fee.
Sell order: You lock base (size in wei). Required value = size. When matched, you receive quote minus fee and the taker receives the base.

Price tick is in basis-point style (e.g. 10000 = 1:1, 9500 = 0.95).

Expiry block: set to 0 for no expiry; otherwise the order becomes invalid after that block.

    Match order (matcher only)
    
        
    Only the configured matcher address can call matchOrder. When matching a buy order: you send fillWei (the base size) and the maker receives that size while the taker receives notional minus fee. When matching a sell order: you send notional (fillWei × priceTick / 10000) and the maker receives notional minus fee while the taker receives fillWei.
